I made this MPF build for a scratch built solid state re-creation of a 1978 Gottlieb Gemini.  
It uses a mini PC, P3-ROC board set and Mission Pinball Framework (MPF) to control EM mechs including a player unit and score motor for sound effect authenticity. 
Playfield used is a self made reproduction, plastics and backglass are old stock. 
Currently programmed for 3 ball play with 5 ball rules (100pt pops, red and yellow stars not linked).
